怎么样去创建动态效果

一、概念

层：是一种HTML内容的容器，层可定位在页面上的任意位置，并且其中可包含文本，图像，表单等所有可直接用于文档的元素

二、层的创建方法

  1. 布局面板 “插入”面板中,在“布局” 选项卡中，选择层(layer)按钮，在编辑区拖动即可插入层
  2. 层用于控制浏览器窗口中对象的精确位置
  3. 在IE中通常为<div></div>或<span></span>
    注： 如一次性插入多个层，可按“CTRL”键画层

层的属性设置

  • 层编号：当前层名称
  • 左，上：相对于页面或其父层左上的位置
  • 高，宽：层的宽度和高度
  • Z轴：设置层的层次属性
  • 可见性：设置层的可见性
    Default：默认是可见的
    inherit: 继承
    Visible：显示层及内容　
    Hidden：隐藏层及内容
  • 背景图像
  • 背景颜色
  • Overflow （溢出）：选择当层内容超出层尺寸时如何处理
    • Visible：自动增加尺寸
    • Hidden：隐藏超出部分内容
    • Scroll：添加滚动条
    • Auto：内容超出后自动添加滚动条
    Clip：选区的L,T,R,B分别设置层的可视区域

选择层

在Layers面板中，单击层名；单击层边框；按shift键，单击层中任意位置；单击层标记。
注：选择多个层：按shift键，点击

建立嵌套层

嵌套层是指包含在其他层中的层，使用嵌套层的好处是确保该层能永远之于其父层的上方

方法：

  1. 新建一个层layer1
  2. 按住alt键不放，在layer1中的任意位置画一个新层layer2
    注：层的关系有嵌套和重叠。重叠两个层是相互独立的；嵌套时子层会随父层变化而变化。

层的剪辑

  1. 新建一个层layer1
  2. 在层中插入一图象
  3. 建立一嵌套层
  4. 移动嵌套层位置和改变大小，是其套住正好要剪辑的图像部分。
  5. 记下嵌套层属性面板中L、T、W、H值。
  6. 选中layer1，在属性面板中clip选区中输入：
    L(刚记下的L)；T(刚记下的T)；
    R(L+W)；B(T+H)。
  7. 单击编辑区空白处。

将层转换为表格:

修改 > 转换 > 层到表格
  1. 最精确:为每个层都创建表格单元格,层与层之间的空间使用单元格代替
  2. 最小合并空白单元:如果层与层之间的距离小于指定的像素宽度,则舍弃层之间的距离
  3. 使用透明的GIF:
    用透明的GIF图象填充表格的最前一行,以确保表格在所有浏览器中以相同的列宽显示
  4. 置于页面中央:
    将转换后的表格在页面中居中显示

三、时间线

是Dreamweaver实现动画的一种方法，用户不需要编辑代码或使用任何ActiveX、Java或其他任何插件即可在网页上制作动画效果。时间线主要是通过改变图层的位置、属性、可见性以及图层的顺序来实现动画效果的。

时间线的编辑:

通过拖动路径创建时间线
步骤:
  1. 创建一个新层,拖动到开始位置;
  2. 打开时间轴面版.选择 修改 > 时间轴 > 录制层路径
  3. 随意拖动层,创建自己想要的路径

内联框架的使用:

内联框架是网页中一种独特的使用方法.和层或表格结合起来使用往往能收到许多意想不到的效果
代码如下:
<iframe></iframe>表示框架

属性包括：

  1. Scrolling：
    滚动条“auto”自动“no”无“yes”有
  2. width (宽）和height（高）
    例：
    <iframe src="1.html" scrolling="auto" width="700" height="200"></iframe>
