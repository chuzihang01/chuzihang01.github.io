一、在网页中插入flash按钮、文本、动画及插件 音乐 视频等。

  1. 插入音乐:
    a.<EMBED SRC=“*…”>这个标记可以放在 <BODY> 和 </BODY> 之间的任何地方。在这个标记放置的位置会出现一个类似媒体播放程式的图案，可以控制播放音乐或暂停。
    <embed src="10.mid" width=145　height=20>
    如果我要把音乐当作背景音乐来用，也就是不要显示控制面板。而且一进入画面就要播放，然後一直重复。
    就要这样写
    <embed　src=“…" hidden=true autostart=true loop=true>
    b.如果我要把音乐当作背景音乐来用，也就是不要显示控制面板。而且一进入画面就要播放，然後一直重复。
    就要这样写
    <embed　src=“…" hidden=true autostart=true loop=true>
    属性：
    1. AUTOSTART=“TRUE / FALSE”是否要在音乐档传完之後，就自动播放音乐。
    2. LOOP=“TRUE / FALSE / 整数”设定是否要自动反覆播放。
    3. WIDTH 和 HIGH=“整数”设定控制面版的宽度和高度
    4. HIDDEN="TRUE" 如果写上这个属性的话，就会完全隐藏控制面板
    <EMBED>可以放在<BODY> 和 </BODY> 之间的任何地方。
    记住：在这个标记放置的位置会出现一个控制面板的图案，你可以视自己的需求，把它放在适当的位置，也可以使用HIDDEN=TRUE 将面板隐藏起来，配合AUTOSTART=TRUE 当背景音乐使用。
  2. 插入背景音乐：
    你如果想插入背景音乐，可以使用以下的格式：
    <bgsound src=“URL”>
    循环播放：
    <bgsound src=“url” loop=“-1”>
    按设定次数播放：
    <bgsound src=“url” loop=“次数” >
  3. 插入视频动画
    1. 基本语法
      <img src=“url.gif” dynsrc=“url.avi”>
      a.用 url.avi 这一 AVI文件来播放视频；
      b.用 url.gif 这一 GIF 图象作为视频的封面，即：在浏览器尚未完全读入 AVI 文件时，先在 AVI 播放区域显示该图象。
    4. 插入flash动画的方法
    1. 单击插入/常用/flash按钮
    2. 插入菜单/媒体/ flash按钮 属性的设置：
      1. menu参数：设置在浏览器中显示的flash电影上，单击右键时，是否显示快捷菜单。
      2. wmode参数：控制电影的透明：如让flash电影背景透时，可在param列中输入wmode,然后在value列中输入transparent
    如果是插入swf文件：有以下几个属性：
    1. wmode=transparent
    2. vspace=“数值” hspace=“数值”（离页面左边和顶部的距离）
    3. align=“left,right”
      注：插入前要首先保存文件

编辑网页的头部内容

  1. 一个HTML文件，通常由包括标记间的头部和包括在标记间的正文组成。
  2. 头部除了标题外，还可以包括很多非常重要的信息。如文档的类型，语言的解码方式、以及引擎的关键字等。

显示与插入头部信息：

  1. View\Head Content(头部信息)命令，显示头部内容
  2. Insert\Head Tags(头部标记)包括：
    a Meta: 元数据
    b Keywords：关键字
    c Description：描述
    d Refresh：刷新
    e Base：基址
    f Link ：链接

插入元数据：

  1. <meta>标记是实现元数据的主要标记，它能够提供文档的关键字、描述等多种信息，
  2. <meta>标记有如下两种书写方式：<meta name=“元数据属性名称” content=“属性值”><meta http-equiv=“元数据属性名称” content=“属性值”>
  • 很多搜索引擎搜索页面时，实际上搜索的是页面里<meta>标记中名为”keywords”(关键字)和”description”(描述)，
  • <meta name=“keywords” content=“合肥信息学院，信息技术，计算机人才">
  • <meta name="description" content="教育，学校">
  • 这样当其他人利用搜索引擎搜索“合肥，信息技术”等字样时，就很可能找到这个页面，从而增大该页面的访问量
  • 设置页面的文字解码方式：页面的文字解码方式可通过页面属性来设置
  • <meta http-equiv=“content-Type” content=“text/html”;charset=“gb2312”>
  • <meta http-equiv="Content-Type" content="text/html; charset=gb2312">
  • <meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1">
  • 设定这是 HTML 文件及其编码语系，中文网页请使用gb2312 ，或者不设编码亦 可，纯英文网页建议使用 iso-8859-1。
  • <meta name=“GENERATOR” content=“Mozilla/4.04 [en] (Win95; I) [Netscape]”>
  • <meta name=“GENERATOR” content=“Microsoft FrontPage 3.0”>
  • 这只表示该网页由什么编辑器写成。

让页面定时刷新或定时跳转

  1. <meta http-equiv=“refresh” content=“5”>每隔5秒就会刷新一次。
  2. 如希望页面经过一段时间跳转另一个地址的页面
    <meta http-equiv="refresh" content="3;url=http://www.sina.com">
    这一行较为实用，能於预定秒数内自动转到指定的网址。原始码中 3 表示 3秒。
